# Cypress

## 上手教程
* 使用cypress，需要安装nodejs。使用npm或yarn安装都行。这里使用npm安装。

```
cd /your/project/path
npm install cypress --save-dev
```
如果是新建的工程，你需要先运行
```
npm init
```
* 启动cypress

可以使用npx执行cypress

```
npx cypress open
```
* 切换浏览器

官网上提到cypress可以找到本机上所有兼容的浏览器。因此可以切换浏览器来执行。
